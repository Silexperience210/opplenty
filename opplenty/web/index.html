signet verrouille
⚠ MAINNET — chaque inscription depense de vrais bitcoins. Un reveal malforme = fonds bloques definitivement. Teste d'abord sur signet.

Encoder

Chaque octet → deux opcodes Tapscript (opcode % 22 == nibble). Aucun push de donnees.

Decoder

Hex brut d'un script ou d'une transaction complete. Cherche le magic 55x7, replie mod 22.

Acces

Mnemonique chiffree sur disque (scrypt + AES-256-GCM). Le mot de passe ne vit qu'en RAM du process.

Reception

Adresse Taproot BIP86. Scanne le QR pour l'alimenter depuis un autre wallet.

Inscription on-chain

Commit (cle interne NUMS, key path indepensable) puis reveal script-path. Roundtrip verifie avant diffusion.
