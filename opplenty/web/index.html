op·plenty
signet verrouillé

Encoder

Chaque octet → deux opcodes Tapscript (opcode % 22 == nibble). Aucun push de données.

Décoder

Hex brut d'un script ou d'une transaction complète. Cherche le magic 55×7, replie mod 22.

Accès

La mnémonique reste chiffrée sur disque (scrypt N=2²⁰ + AES-256-GCM). Le mot de passe ne vit qu'en RAM.

Réception

Adresse Taproot BIP86. Alimente-la (faucet signet pour tester).

Inscription on-chain

Commit (clé interne NUMS, key path indépendable) puis reveal script-path. Roundtrip vérifié avant diffusion.
